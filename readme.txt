1. `bash_aliases' and `bashrc' are used for Bash Shell.
    `astylerc' is used for Artistic Style, a C/C++ code formatter.

    You can execute the fellowing commands in the $HOME

        ln -s utils/bash_aliases    .bash_aliases
        ln -s utils/bashrc          .bashrc
        ln -s utils/git-completion.bash     .git-completion.bash
        ln -s utils/astylerc        .astylerc
